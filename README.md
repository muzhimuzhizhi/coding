1. 负责公司平台产品的基础架构的设计与实现；

 1. 7年以上互联网开发经验，有大规模后端服务的设计和开发方面经验，对 Unix/Linux 有深入了解；

 5. 熟悉常用的算法，熟悉开源框架，解决问题能力强；
 6. 对分布式计算，分布式存储的架构设计理念和动机有一定理解；
